# zudio-analysis
Interactive Excel KPI dashboard analyzing Zudio (Tata Group) retail sales across 8 Indian states and 5 cities. Tracked revenue by state (top: West Bengal , Tamil Nadu) , gender split (Kids 34%, Men 33%, Women 33%), age-wise demand, and operating hour patterns. Built with 8 chart types,15 worksheets, and dynamic State, City, Category, Month slicers.
